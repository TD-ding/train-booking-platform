# 管理端文档

管理端是前台应用的一部分，通过路由守卫和角色判断控制访问。管理员和普通用户使用同一前端应用，管理员登录后导航栏出现「管理后台」入口。

## 访问权限

- 所有 `/admin/*` 路由需要 JWT 中 `role === 'admin'`
- 非管理员访问管理页面自动跳转首页
- 管理员也拥有普通用户的所有功能（购票、订单、联系人等）

## 默认管理员账号

种子数据中预置了管理员账号：

| 字段 | 值 |
|------|------|
| 用户名 | admin |
| 密码 | admin123 |
| 角色 | admin |

> **生产环境请立即修改此密码。**

## 管理功能概览

### 仪表盘 `/admin`

展示平台核心运营指标。

**KPI 卡片：**

| 指标 | 说明 |
|------|------|
| 总订单数 | 所有状态订单的合计 |
| 总收入 | 已支付 + 已完成订单的金额合计 |
| 注册用户 | 普通用户数量 |
| 待处理订单 | 状态为 pending 的订单 |
| 今日订单 | 当天创建的订单 |

**图表：**

- **7 天销售趋势**：SVG 柱状图，展示近 7 天每日订单数和收入
- **线路分布**：SVG 环形图，展示 Top 10 线路的订单占比

**数据列表：**

- **热门线路 Top 5**：按订单量排序的线路列表
- **最近订单**：最新 10 条订单的简要信息

---

### 车次管理 `/admin/trains`

#### 车次列表

表格展示所有车次，列包括：车次号、类型、出发站、到达站、发车时间、到达时间、历时、状态（运行中/停运）、操作按钮。

#### 添加车次

点击「添加车次」按钮，弹出对话框填写：

| 字段 | 必填 | 说明 |
|------|------|------|
| 车次号 | 是 | 唯一标识，如 G1、D3108 |
| 类型 | 是 | 高铁 / 动车 / 普快 |
| 出发站 | 否 | 下拉选择 |
| 到达站 | 否 | 下拉选择 |
| 发车时间 | 否 | 如 08:00 |
| 到达时间 | 否 | 如 12:00 |
| 历时 | 否 | 如 4小时0分 |
| 状态 | 是 | 运行中(active) / 停运(inactive)，默认运行中 |

#### 编辑车次

点击行内编辑按钮，弹出预填表单。

#### 删除车次

- 如果该车次有未完成订单（pending/paid），会弹出错误提示拒绝删除
- 删除车次会级联删除其经停站和座位库存数据

#### 时刻表管理

点击行内时钟图标，弹出时刻表管理对话框：

- 查看该车次所有经停站（按顺序排列，显示到达/发车时间）
- 添加经停站：选择车站、设置顺序、到达/发车时间
- 编辑经停站：修改车站、时间等
- 删除经停站

经停站用于：
1. 搜索时判断车次是否经过用户选择的出发站和到达站
2. 计算座位库存适用的区间

### 车站管理 `/admin/stations`

表格展示所有车站。

#### 车站字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | 是 | 车站名称，如"北京南" |
| 编码 | 是 | 唯一编码，如"BJN" |
| 城市 | 是 | 所属城市 |
| 拼音 | 否 | 用于前端搜索匹配 |

支持完整的 CRUD 操作。编码重复时会提示「车站编码已存在」。

---

### 订单管理 `/admin/orders`

展示所有用户的订单。

**功能：**
- 按状态筛选（全部/待支付/已支付/已取消/已退票/已完成）
- 表格展示：订单号、用户、车次、座位类型、乘客、出发站→到达站、金额、状态、下单时间
- **退票操作**：对已支付的订单执行退票
  - 订单状态变为 `refunded`
  - 支付状态变为 `refunded`
  - 对应区间座位库存 +1

---

### 用户管理 `/admin/users`

展示所有用户列表。

**功能：**
- 查看用户信息：ID、用户名、真实姓名、手机、邮箱、角色、注册时间
- **切换角色**：在 `user` 和 `admin` 之间切换
  - 将普通用户提升为管理员
  - 将管理员降级为普通用户（注意不要把自己降级）

---

## 管理端 API 汇总

所有管理端 API 前缀为 `/api/admin`，需要 JWT 中 role 为 admin。

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /dashboard | 仪表盘统计数据 |
| GET | /users | 用户列表 |
| PUT | /users/:id/role | 修改用户角色 |
| GET | /trains | 车次列表 |
| POST | /trains | 创建车次 |
| PUT | /trains/:id | 更新车次 |
| DELETE | /trains/:id | 删除车次 |
| POST | /trains/:id/stops | 添加经停站 |
| PUT | /trains/:trainId/stops/:stopId | 更新经停站 |
| DELETE | /trains/:trainId/stops/:stopId | 删除经停站 |
| GET | /stations | 车站列表 |
| POST | /stations | 创建车站 |
| PUT | /stations/:id | 更新车站 |
| DELETE | /stations/:id | 删除车站 |
| GET | /orders?status= | 订单列表 |
| POST | /orders/:id/refund | 退票 |
